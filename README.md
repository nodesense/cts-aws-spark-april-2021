# to install Anaconda 3.7 along with 3.8

Open Anaconda prompt

```
conda create -n py37 anaconda=2020.07 python=3.7

When prompted to proceed, say Yes
```

----

open anaconda prompt

```
conda activate py37

pyspark
```




Whenever we need to deactivate / (most of the time we don't need)

```
conda deactivate
```
